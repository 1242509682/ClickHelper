## 📖 项目简介

点击助手是一款 Windows 桌面自动化辅助工具，支持鼠标点击、键盘按键模拟、**宏录制与回放**等强大功能。用户可自由组合位置列表（点击/按键序列）、设置循环次数与间隔，并支持定时任务、全局热键、脚本保存与加载。新增的**宏录制器**可捕获鼠标移动、点击、滚轮及键盘操作，生成可编辑的宏文件，并能一键转换为位置列表，极大提升自动化效率。

程序基于 .NET 10.0 和 Windows Forms 开发，界面简洁，支持托盘常驻及单实例运行。

## ✨ 主要功能

### 1. 位置列表管理
- 记录鼠标坐标（X, Y），可添加描述。
- 每个位置独立设置操作类型（鼠标点击 / 键盘按键）。
- 鼠标支持左键、中键、右键；键盘支持任意键（含 Ctrl/Shift/Alt 修饰键）。
- 延迟执行单个坐标。
- 操作模式：单击（按下+弹起）、仅按下、仅弹起。
- 列表支持添加、删除、上移/下移、编辑，变动自动保存。

### 2. 循环控制
- 循环次数：`-1` 无限循环，正整数为执行轮数，`0` 不执行。
- “批量执行”模式：每轮一次性执行全部位置（非顺序逐个执行）。

### 3. 间隔时间
- 自定义每次点击/按键之间的间隔（`1ms` ~ `30分钟`），单位毫秒。

### 4. 热键控制
- **启动/停止热键**：可自定义按键（F1~F12，数字，字母 A~Z）。
- **记录位置热键**：`Alt + 自定义按键`，快速将当前鼠标位置加入列表。
- **宏录制切换热键**：默认 `F6`，按一次开始录制，再按停止录制。
- **宏播放切换热键**：默认 `F7`，按一次播放最新宏，再按停止播放。

### 5. 宏录制与回放（✨ 新增）
- **录制**：捕获鼠标移动、左/中/右键点击、滚轮滑动、键盘按键（支持组合键），精确记录相对时间偏移。
- **保存**：宏以 `.txt` 文本格式存储（每行一条指令），便于手动编辑或版本管理。
- **管理**：内置“宏管理器”，以双列表形式展示所有宏文件及其指令，支持：
  - 新建、保存、加载、删除宏文件。
  - 清空宏文件列表。
  - 编辑单个宏指令（动作、时间、坐标、键码、滚轮增量）。
  - 拖拽调整宏指令顺序。
  - 实时监听宏文件夹变化，自动刷新文件列表。
- **转换**：一键将当前宏转换为位置列表，转换后可直接用于顺序/批量执行，实现录制与点击任务的融合。
- **播放**：支持宏回放，可随时暂停/停止，并自动同步主窗口状态（显示“播放中...”）。

### 6. 定时任务
- 可启用/禁用定时，自定义开始时间（含日期时间）和结束时间（可选“无结束”）。
- 在有效时间范围内，定时自动启动任务；超出时间范围自动停止。
- 若位置列表为空或循环次数为 0，定时启动时会自动禁用并提示。

### 7. 配置管理
- **保存配置**：将当前所有配置（位置列表、热键、间隔、循环、定时等）保存为 `.json` 文件。
- **加载配置**：从指定 `.json` 文件恢复配置，并同步更新界面和热键。

### 8. 托盘功能
- 程序启动后托盘.常驻，即使主窗口关闭也不会退出（可右键托盘退出）。
- 点击最小化按钮时，主窗口隐藏至托盘，双击托盘图标恢复。
- 支持自动最小化：勾选“自动最小化”后，执行任何任务（启动点击、录制、播放）时自动隐藏窗口。

### 9. 实时状态反馈
- 主窗口状态栏显示当前状态（空闲、运行中、录制中、播放中等）。
- 鼠标坐标实时显示于状态栏右侧。
- 新增按键显示：按下任意键时，状态栏会临时显示按下的键名（如“A”、“F1”等）。

### 10. 单实例运行
- 程序同一时刻只允许一个实例运行，重复启动会弹出提示并退出。

## 🚀 操作说明

### 基本流程（位置列表模式）
1. **添加位置**：点击“位置列表”按钮，在编辑窗口中添加坐标、操作类型、按键、模式等；也可使用热键 `Alt + 自定义键` 快速记录当前位置。
2. **设置循环次数和间隔**：调整“循环”数值框（-1 无限，>0 轮次）和“间隔”数值框（单位毫秒）。
3. **启动/停止**：点击“启动”/“停止”按钮，或按热键（默认 F9）。
4. **定时任务**：点击“定时管理”设置时间段，启用后自动运行。

### 宏录制与回放
1. **录制宏**：
   - 点击“宏管理器”按钮打开管理器。
   - 点击“录制”按钮（或按热键 F6），执行任意鼠标/键盘操作。
   - 点击“停录”（或再次按 F6）结束录制，宏自动保存到 `Macros` 文件夹，文件名包含时间戳。
2. **管理宏**：
   - 在宏管理器中，左侧上方显示所有宏文件，下方显示选中宏的指令列表。
   - 支持新建、保存、加载、删除宏文件，以及清空所有宏文件。
   - 双击指令可编辑，拖拽可调整顺序。
3. **播放宏**：
   - 在管理器中选择一个宏，点击“播放”（或按热键 F7）开始回放；再次按 F7 可停止播放。
   - 主界面也提供“播放”按钮（播放最新宏）和“宏管理器”入口。
4. **转换为位置列表**：选中一个宏，点击“转换”按钮，将宏中的所有点击/按键转为位置列表，追加到现有列表，供常规模式使用。

### 自动最小化
- 勾选主窗口的“自动最小化”复选框后，点击“启动”、“录制”或“播放”时，主窗口会自动隐藏到托盘，保持界面清爽。

## 📁 配置文件与宏文件

- **主配置文件**：`Config.json`（位于 exe 同级目录），记录所有设置（位置列表、热键、定时等）。
- **宏文件**：存放在 `Macros` 文件夹，扩展名为 `.txt`，每行一条指令，格式为 `时间(ms) 动作 参数...`。例如：
```
0 Move 100 200
150 LDown 100 200
250 LUp 100 200
300 KDown 65
400 KUp 65
```
支持动作：`Move`, `LDown`, `LUp`, `RDown`, `RUp`, `MDown`, `MUp`, `Wheel`, `KDown`, `KUp`。
用户可自行编辑或备份。

## ⚠️ 注意事项

- 本工具仅用于合法自动化操作，请遵守相关软件的使用协议。
- **管理员权限**：宏录制与回放使用了全局钩子，需要以管理员身份运行程序，否则可能无法捕获跨进程事件。
- 键盘组合键设置时，请确保选中的按键未被系统或其他程序占用。
- 宏录制时会自动排除配置中的热键（启动热键、记录热键、宏热键），避免干扰。
- 位置列表中的“按下”和“弹起”模式需配对使用，避免鼠标/键盘状态异常。
- 托盘图标常驻，如需完全退出，请右键托盘图标选择“退出程序”，或直接关闭主窗口（会同时退出进程）。
