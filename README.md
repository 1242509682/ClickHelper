## 📖 项目简介

点击助手是一款 Windows 桌面自动化辅助工具，主要用于模拟鼠标点击和键盘按键操作。用户可自定义位置列表（含坐标、操作类型、按键模式）、循环次数、执行间隔，并支持定时任务、热键控制、脚本保存与加载等功能，适用于游戏辅助、办公自动化等场景。

程序基于 .NET 10.0 和 Windows Forms 开发，界面简洁，占用资源少，支持托盘隐藏运行。

---

## ✨ 主要功能

- **位置列表管理**  
  - 记录鼠标坐标（X, Y），可添加描述。  
  - 每个位置可独立设置操作类型（鼠标点击 / 键盘按键）。  
  - 鼠标支持左键、中键、右键，键盘支持任意键（含 Ctrl/Shift/Alt 修饰键）。  
  - 每个位置可设置操作模式：单击（按下+弹起）、仅按下、仅弹起。  
  - 支持列表的添加、删除、上移/下移、编辑，列表变动自动保存。

- **循环控制**  
  - 可设置循环次数：`-1` 无限循环，正整数为执行轮数，`0` 表示不执行。  
  - 支持“同时执行”模式：每轮一次性执行全部位置（而非顺序逐个执行）。

- **间隔时间**  
  - 自定义每次点击/按键之间的间隔（`1ms` ~ `30分钟`），单位毫秒。

- **热键控制**  
  - **启动/停止热键**：可自定义按键（F1~F12，数字，字母 A~Z）。  
  - **记录位置热键**：`Alt + 自定义按键`，快速将当前鼠标位置加入列表。

- **定时任务**  
  - 可设置启用/禁用定时。  
  - 自定义开始时间（含日期时间）和结束时间（可选“无结束”）。  
  - 在有效时间范围内，定时自动启动任务；超出时间范围自动停止。  
  - 若位置列表为空或循环次数为 0，定时启动时会自动禁用并提示。

- **脚本管理**  
  - **保存脚本**：将当前所有配置（位置列表、热键、间隔、循环、定时等）保存为 `.json` 文件。  
  - **加载脚本**：从指定 `.json` 文件恢复配置，并同步更新界面和热键。

- **托盘功能**  
  - 点击最小化按钮时，程序隐藏至系统托盘。  
  - 双击托盘图标恢复主窗口；右键托盘菜单可显示窗口或退出程序。

- **鼠标位置实时显示**  
  - 主窗口底部实时显示当前鼠标坐标，便于定位。

## 🚀 操作说明

1. **启动程序后，首先添加位置**  
   - 点击 **“位置列表...”** 按钮，打开编辑窗口。  
   - 点击 **“添加”** 按钮，会弹出编辑对话框，设置坐标、操作类型、按键、模式等。  
   - 也可以使用热键（`Alt + 自定义键`）快速将当前鼠标位置加入列表（默认左键单击）。

2. **设置循环次数和间隔时间**  
   - **“循环”** 数值框：`-1` 无限，正整数为执行轮数，`0` 不执行。  
   - **“间隔”** 数值框：单位毫秒。

3. **启动任务**  
   - 点击 **“启动”** 按钮，或按热键（默认 `F9`）。  
   - 任务会按照位置列表顺序（或同时执行）循环执行，直到完成指定轮数或手动停止。

4. **停止任务**  
   - 点击 **“停止”** 按钮，或再次按热键。

5. **定时设置**  
   - 点击 **“定时设置...”** 按钮，在弹出的窗口中勾选 **“启用定时”**。  
   - 设置开始时间和结束时间（可勾选 **“无结束”**）。  
   - 确定后，程序会每秒检查当前时间，自动在有效时间段启动/停止任务。

6. **保存/加载脚本**  
   - 点击 **“保存脚本”** 选择路径保存 `.json` 文件。  
   - 点击 **“加载脚本”** 选择 `.json` 文件恢复所有配置。

## 📁 配置文件

程序运行后会在 `exe` 同目录下生成 `Config.json` 文件，记录所有配置（位置列表、热键、间隔、循环、定时等）。用户可直接编辑该文件（JSON 格式），但不建议手动修改，以免格式错误导致加载失败。

脚本文件（`.json`）与 `Config.json` 结构一致，可互相通用。

## ⚠️ 注意事项

- 本工具仅用于合法的自动化操作，请遵守相关软件的使用协议。  
- 使用键盘组合键时，请确保选中的按键未被系统或其他程序占用。  
- 定时任务的结束时间若设为“无结束”，则任务在开始后将持续运行，直到手动停止或程序退出。  
- 位置列表中的 **“按下”** 和 **“弹起”** 模式可组合实现拖拽、长按等复杂操作，但需注意配对使用，否则可能导致鼠标或键盘状态异常。  
- 程序最小化后隐藏至托盘，不会在任务栏显示；如需完全退出，请右键托盘图标选择 **“退出程序”**。  
- 若位置列表为空或循环次数为 0，启动操作会弹窗提示，定时任务也会自动禁用。
