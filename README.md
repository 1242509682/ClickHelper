## 📖 项目简介

点击助手是一款 Windows 桌面自动化辅助工具，集成了**鼠标点击模拟**、**键盘按键控制**、**图像与文字识别点击**、**宏录制与回放**、**定时任务**等丰富功能。程序基于 .NET 10.0 和 Windows Forms 开发，界面清爽，支持托盘常驻及单实例运行。

---

## 🎯 v1.0.5 更新亮点

本次更新是 **OCR 引擎全面重构** 与 **内存性能大幅优化** 的里程碑版本。

### 🔥 核心改进

| 改进项 | v1.0.4 | v1.0.5 |
|--------|--------|--------|
| **OCR 引擎** | PaddleOCRSharp（Paddle 格式） | **RapidOCRSharpOnnx（ONNX 格式）** |
| **推理后端** | Paddle 原生推理 | **ONNX Runtime（CPU 执行提供者）** |
| **模型依赖** | ~420 MB | **~142 MB（↓ 66%）** |
| **内存占用** | 200-600 MB（不回落） | ** < 100 MB（稳定）** |
| **推理速度** | 较慢 | **提升 4-5 倍** |

### ✨ 新增与优化

- **全新 OCR 引擎**：从[PaddleSharp](https://github.com/sdcb/PaddleSharp) 迁移至 [RapidOCRSharpOnnx](https://github.com/meloht/RapidOCRSharpOnnx)，采用 ONNX Runtime 推理，速度更快，内存占用更低。
- **图像内存池化**：引入 `MatPool` 对象池，复用 [OpenCvSharp](https://github.com/shimat/opencvsharp) `Mat` 对象，大幅减少内存分配。
- **文字识别缓存**：相同图片二次识别毫秒级响应，零内存开销。
- **坐标编辑器重构**：
  - 合并“文本输入”与“组合键”为统一的“输出内容”文本框。
  - 新增“初始化OCR”和“识别文字”两个独立按钮，状态实时反馈。
  - 阈值/置信值增加中文说明提示，方便调参。
- **OCR 结果展示**：识别结果自动按行分隔，弹窗高度自适应，支持一键复制。
- **FlaUI 后台输入**：支持向记事本等控件后台输入文本，不移动鼠标、不激活窗口。
- **定时任务修复**：宏播放改为异步执行，解决阻塞线程问题。
- **依赖管理**：程序启动前自动检测依赖，引导用户下载安装，初始化完成后自动重启生效。
- **调试日志**：增加 `Logger.Debug` 模块，支持运行时日志追踪，便于问题排查。

## ✨ 主要功能

### ✦ 点击核心
- **四种点击类型**：鼠标点击、键盘按键、文本输入、组合键。
- **UIA（UI Automation）支持**：不移动鼠标对桌面控件执行点击。
- **坐标增强**：支持坐标重选，(0,0) 自动使用当前鼠标位置。

### ✦ 图像与文字识别点击
- **OpenCvSharp 模板匹配**：高效图像识别，CPU 占用低。
- **OCR 文字识别**：集成 RapidOCRSharpOnnx（PP-OCRv6），识别屏幕文字并执行点击。
- **图片预览增强**：滚轮缩放、双击还原、右键识别文字/复制/另存为/复制 Base64。

### ✦ 宏录制与回放
- 捕获鼠标移动、点击、滚轮、键盘按键（含组合键）。
- 宏以 `.txt` 文本格式存储，支持手工编辑。
- 内置宏管理器：新建、保存、加载、删除、编辑、拖拽排序、一键转位置列表。

### ✦ 定时任务
- **日期模式**（指定起止时间）和 **计时模式**（设定运行时长）。
- 执行目标可选：**位置列表** 或 **宏播放**。
- 独立热键（默认 F8）一键启用/停用定时。

### ✦ 位置列表（坐标管理）
- 支持鼠标点击、键盘按键、文本输入、组合键四种类型。
- 独立设置按键、修饰键（Ctrl/Shift/Alt）、操作模式（单击/按下/弹起）、延迟时间。
- 列表支持添加、删除、上移/下移、复制、编辑，变动自动保存。

### ✦ 循环与间隔控制
- 循环次数：`-1` 无限，正整数为轮数，`0` 不执行。
- 间隔时间：自定义每次操作之间的延迟（1ms ~ 30分钟）。
- “批量执行”模式：一次性执行所有位置，而非逐个循环。

### ✦ 热键体系
| 功能 | 默认热键 | 说明 |
|------|---------|------|
| 启动/停止点击 | F9 | 可自定义 |
| 记录坐标 | Alt + 自定义键 | 快速记录当前鼠标位置 |
| 宏录制 | F6 | 开始/停止录制 |
| 宏播放 | F7 | 播放/停止宏 |
| 截图 | Alt + A | 框选区域生成图像模板 |
| 定时切换 | F8 | 启用/停用定时任务 |

### ✦ 截图与图像匹配
- 按下截图热键进入全屏选区模式，左键拖拽选择区域，按 Enter 确认。
- 截图自动转为图像模板，弹出编辑器调整参数。
- 通过 OpenCvSharp 高效匹配，实现基于图像的智能点击。

### ✦ OCR 文字识别
- 截图模式下按 `B` 键，对框选区域进行 OCR 识别，结果可复制。
- 首次使用需下载模型及依赖，程序自动引导。
- 主窗口状态栏显示 OCR 就绪状态。

### ✦ 配置管理
- 主配置保存在 `Config.json` 中（位置列表、热键、间隔、循环、定时等）。
- 支持“保存配置”和“加载配置”，便于切换不同场景。

### ✦ 界面与交互优化
- 自定义标题栏，支持拖拽移动，圆角窗口（半径16px），渐变背景。
- 状态栏实时显示运行状态、鼠标坐标、OCR 状态。
- 托盘常驻，双击恢复主窗口，右键退出程序。
- 自动最小化：勾选后启动点击/录制/播放时自动隐藏至托盘。

## 🚀 快速使用指南

### 1. 基础点击
1. 打开“坐标管理”窗口，添加坐标（可手动或“重选”拾取）。
2. 设置循环次数和间隔。
3. 按“启动”或热键（默认 F9）开始执行。

### 2. 图像点击
1. 在坐标编辑器中勾选“图像识别” → 重新截图或加载图片。
2. 调整匹配阈值（0~1）。
3. 可选：勾选“文字点击”并填入目标文字，实现 OCR 辅助点击。
4. 保存后执行即可。

### 3. 宏录制与回放
1. 打开宏管理器，点击“录制”或按 F6。
2. 执行操作后再次点击“停录”或按 F6。
3. 自动生成新宏，在管理器中可编辑、播放（F7）或转换为位置列表。

### 4. 定时任务
1. 点击“定时管理”，选择日期/计时模式，设置时间参数。
2. 选择执行目标（位置列表或指定宏）。
3. 按定时热键（默认 F8）启用，再次按则停用。

## 📁 文件说明

| 路径 | 说明 |
|------|------|
| `Config.json` | 主配置文件（位于 exe 同级目录） |
| `Macros/` | 宏文件存储目录（`.txt` 格式） |
| `inference/` | OCR 模型文件夹（需自行下载[PP-OCRv6](https://www.modelscope.cn/collections/PaddlePaddle/PP-OCRv6)，含 `det.onnx` + `rec.onnx`） |
| `Log/` | 异常日志（按日期分文件） |
| `OcrCache/` | 文字识别缓存（JSON 格式，相同图片秒级响应） |

- 本工具仅供合法自动化操作，请遵守相关软件使用协议。
