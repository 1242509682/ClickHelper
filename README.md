## 📖 项目简介

点击助手是一款 Windows 桌面自动化辅助工具，集成了**鼠标点击模拟**、**键盘按键控制**、**图像与文字识别点击**、**宏录制与回放**、**定时任务**等丰富功能。程序基于 .NET 10.0 和 Windows Forms 开发，界面清爽，支持托盘常驻及单实例运行。

---

## ✨ 主要功能（基于 v1.0.4）

### ✦ 点击核心优化
- **新增点击类型**：
  - **文本输入**：直接向焦点窗口输入任意文字（相当于帮你打字）。
  - **组合键**：同时按下多个按键（如 `Ctrl+C`），支持自定义组合。
- **UIA（UI Automation）支持**：可在不移动鼠标的情况下对目标控件执行点击（对游戏进程无效，但适用于桌面应用）。
- **坐标操作增强**：
  1. 支持**坐标重选**（通过“重选”按钮进入全屏拾取）。
  2. 当坐标设为 `(0,0)` 时，自动使用**当前鼠标位置**作为点击坐标。

### ✦ 图像点击优化
- **采用 OpenCvSharp 降低 CPU 占用**：相比传统逐像素搜索，性能大幅提升。
- **支持从本地路径加载图片**：可直接将磁盘上的图片文件作为模板，省去截图步骤。
- **OCR 文字识别点击**：
  - 集成 PaddleOCR v6 模型，可识别屏幕区域文字，并根据文字内容执行点击。
  - 内存占用较高，建议在“关于”窗口勾选“自动加载OCR”方便使用，也可在坐标编辑器中手动点击“释放OCR”再“初始化OCR”以控制内存。
- **加强图片预览窗口**：
  1. 滚轮缩放图片。
  2. 双击还原原始尺寸。
  3. 右键菜单提供“识别文字”、“复制剪贴板”、“另存为文件”等功能。
  4. 支持复制图片的 Base64 字符编码，便于分享或存储。

### ✦ 宏录制与回放
- 支持捕获鼠标移动、点击（左/中/右键）、滚轮、键盘按键（含组合键），精确记录时间偏移。
- 宏以 `.txt` 文本格式存储，便于手工编辑或版本管理。
- 内置宏管理器：支持新建、保存、加载、删除宏，双击编辑指令，拖拽调整顺序，一键转换为位置列表。

### ✦ 定时任务
- 支持**日期模式**（指定起止时间）和**计时模式**（设定运行时长）。
- 可自由选择执行目标：**位置列表**或**宏播放**。
- 独立热键（默认 F8）一键启用/停用定时，运行期间自动锁定配置界面。

### ✦ 位置列表（坐标管理）
- 每条记录支持鼠标点击、键盘按键、文本输入、组合键四种类型。
- 可独立设置按键、修饰键（Ctrl/Shift/Alt）、操作模式（单击/按下/弹起）、延迟时间。
- 列表支持添加、删除、上移/下移、复制、编辑，变动自动保存。

### ✦ 循环与间隔控制
- 循环次数：`-1` 无限，正整数为轮数，`0` 不执行。
- 间隔时间：自定义每次操作之间的延迟（1ms ~ 30分钟）。
- 支持“批量执行”模式：一次性执行所有位置，而非逐个循环。

### ✦ 热键体系
- **启动/停止热键**：默认 F9，可自定义。
- **记录坐标热键**：`Alt + 自定义键`，快速记录当前鼠标位置（伴随动画反馈）。
- **宏录制热键**：默认 F6。
- **宏播放热键**：默认 F7。
- **截图热键**：默认 `Alt + A`，框选区域自动生成图像匹配模板。
- **定时切换热键**：默认 F8。

### ✦ 截图与图像匹配
- 按下截图热键后进入全屏选区模式，左键拖拽选择区域，按 Enter 确认。
- 截图自动转为图像模板，并弹出编辑器供调整坐标、操作类型、匹配阈值、OCR 文字匹配等参数。
- 执行时通过 OpenCvSharp 高效匹配，实现基于图像的智能点击。

### ✦ OCR 文字识别
- 在截图模式下按 `B` 键，对框选区域进行 OCR 识别，结果可复制到剪贴板。
- 首次使用需下载 PaddleOCR v6 模型及依赖（`inference` 文件夹），程序会自动引导下载。
- 主窗口状态栏显示 OCR 加载状态，支持“关于”窗口中的“自动加载OCR”选项。

### ✦ 配置管理
- 主配置保存在 `Config.json` 中（含位置列表、热键、间隔、循环、定时等）。
- 支持“保存配置”和“加载配置”，便于切换不同场景。

### ✦ 界面与交互优化
- 自定义标题栏，支持拖拽移动，内置最小化/关闭按钮。
- 圆角窗口（半径16px），渐变背景，按钮配色区分功能（蓝/橙/绿/紫）。
- 状态栏实时显示运行状态、鼠标坐标、OCR 加载状态。
- 托盘常驻，双击恢复主窗口，右键可退出程序。
- 自动最小化：勾选后，启动点击/录制/播放时自动隐藏至托盘。

## 🚀 快速使用指南

### 1. 基础点击（位置列表模式）
1. 打开“坐标管理”窗口，添加坐标（可手动输入或使用“重选”拾取）。
2. 设置循环次数和间隔。
3. 按“启动”或热键（默认 F9）开始执行。

### 2. 图像点击
1. 在坐标编辑器中勾选“截图”并重新截图或加载图片。
2. 调整匹配阈值（0~1）。
3. 可选：勾选“识文字”并填入目标文字，实现 OCR 辅助点击。
4. 保存后执行即可。

### 3. 宏录制与回放
1. 打开宏管理器，点击“录制”或按 F6。
2. 执行操作后再次点击“停录”或按 F6。
3. 自动生成新宏，在管理器中可编辑、播放（F7）或转换为位置列表。

### 4. 定时任务
1. 点击“定时管理”，选择日期/计时模式，设置时间参数。
2. 选择执行目标（位置列表或指定宏）。
3. 按定时热键（默认 F8）启用，再次按则停用。

## 📁 文件说明

- **Config.json**：主配置文件，位于 exe 同级目录。
- **Macros/**：存放宏文件（`.txt` 格式）。
- **inference/**：OCR 模型文件夹（需自行下载）。
- **Log/**：异常日志（按日期分文件）。
- **OcrCache/**：文字识别点击缓存。
